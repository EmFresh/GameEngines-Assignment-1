# GameEngines Assignment 1
## Description
this is a puzzle platformer type game where your goal is to make your way to the end as fast as posable. This prototype has platforms, start points, end points and checkpoints.

## Controls 
AWSD  - Movement\
Aero Keys - rotation\
ctrl + Z - Undo\
ctrl + Y - Redo
